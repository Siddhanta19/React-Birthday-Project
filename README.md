# Usage :

> ## `npm install`

> Installs the required dependencies

<hr>

> ## `npm start`
>
> Runs the app in the development mode.<br>
> Open [http://localhost:3000](http://localhost:3000)

<hr>

> ## `npm run build`

> Builds the app for production to the `build` folder.<br>
